# Movie Board

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app). It have the following dependecies : 



### React Router [https://reactrouter.com/web/guides/quick-start]

React Router give us the ability to render React Components as routes and helps to manage the anthentication with private routes.



### Axios [https://www.npmjs.com/package/axios]

Axios is used to handle HTTP requests.



### Fuse.js [https://fusejs.io/]

Fuse.js is a search-library. It returns results approximately equals to the user requests rather than exactly.



### Styled-Components[https://styled-components.com/]

Styled-components give us the ability to use some ES6 features in CSS.



### Framer Motion[https://www.framer.com/motion/]

A production-ready motion library for React based on Framers prototyping Software. It's shipped with animation's physics and can be used by a JavaScript component.



### Google Firebase[https://firebase.google.com/]

Google Firebase is used to handle authentication is this project.

# Before you start

This project cannot be started without a .env file at the project's root, is have to include the following informations : 

` REACT_APP_FIREBASE_KEY=""`

`REACT_APP_FIREBASE_DOMAIN=""`

`REACT_APP_FIREBASE_DATABASE=""`

`REACT_APP_FIREBASE_PROJECT_ID=""`

`REACT_APP_FIREBASE_STORAGE_BUCKET=""`

`REACT_APP_FIREBASE_SENDER_ID=""`

`REACT_APP_TMDB_KEY=""`
